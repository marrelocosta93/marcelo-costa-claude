# Skill: UX Review — Usa Ralph Loop

## Checklist
- **Fluxos**: muitos cliques? Ordem lógica? Confirmação em ação destrutiva? Desfazer?
- **Acessibilidade**: contraste WCAG AA 4.5:1? Teclado? Screen reader? Focus visible?
- **Consistência**: botões iguais = ações iguais? Terminologia uniforme?
- **Feedback**: erros ajudam resolver? Loading states? Empty states? Success states?
- **Mobile**: funciona 375x667? Touch targets ≥ 44px? Texto legível? Forms não cortados?
