## Como Usar

- Playbooks sao **referencias estrategicas**, nao scripts rigidos
- Adapte ao contexto do projeto e da equipe
- Combine playbooks conforme necessario (ex: SDD + Database First)
- Skills de workflow implementam partes dos playbooks na pratica
